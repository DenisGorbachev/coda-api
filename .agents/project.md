# Coda API project

## General

- Do not modify `src/lib.rs` (it is autogenerated)
- Disregard the warnings emitted from `src/lib.rs` (it is autogenerated)

## Concepts

### DocData

- Must have fields:
  - `metadata: DocMetadata`
  - `rows: BTreeMap<TableId, Vec<RichRow>>`

### Client::get_doc_data

- Must have parameters:
  - `doc_id: &DocId`
- Must return `DocData`
- Must call `self.get_doc_metadata`
- Must download rows from every base table

Notes:

- Must not download view rows because they duplicate or filter base-table rows

### DocDataCli

- Must derive `Parser`
- Must have fields:
  - `api_key: String` (`#[arg(short, long, env = "CODA_API_KEY", hide_env_values = true)]`)
  - `doc_id: DocId` (`#[arg(short, long, env = "CODA_DOC_ID")]`)
- Must have a `run` method:
  - Must create `Client`
  - Must call `client.get_doc_data`
  - Must write the returned `DocData` to `stdout` as pretty JSON

### examples/doc_data.rs

- Must define `main`
  - Must parse `DocDataCli`
  - Must call `DocDataCli::run`
